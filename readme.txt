== Description ==

Complete dropdown tree navigation that contain pages, categories with posts, tags, authors and own menu.

= Features and highlights =
* Whole site in one place
* Excluding pages and authors from navigation
* Simple widget
* 4 skins

= Translations/Languages =
This plugin is available in your Wordpress language on site and in English and Polish in Dashboard.

= Skins =
* iOS Lion
* Windows 8
* Windows 7
* Windows XP

== Installation ==

1. Unpack and upload easytree.zip to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add a EasyTree widget to your theme, or use `[easytree]` shortcode on any page or post.

== Frequently Asked Questions ==

= Where can I change plugin settings? =

Go to Dashboard -> Settings -> EasyTree
